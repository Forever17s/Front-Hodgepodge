### Git & SVN

| Git                                    | SVN                                |
| -------------------------------------- | ---------------------------------- |
| Git 是一个 **分布式** 的版本控制工具   | SVN 是 **集中式** 版本控制工具     |
| 它属于第 3 代版本控制工具              | 它属于第 2 代版本控制工具          |
| 客户端可以在其本地系统上克隆整个存储库 | 版本历史记录存储在服务器端存储库中 |
| 即使离线也可以提交                     | 只允许在线提交                     |
| `push` / `pull` 操作更快               | `push` / `pull` 操作较慢           |
| 工程可以用 `commit` 自动共享           | 没有任何东西自动共享               |
